# MERN Todo App

A full-stack Todo app built using **MongoDB, Express, React, and Node.js**.

## Features
✅ Add, Edit, Delete Todos  
✅ Responsive UI  

## Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js, MongoDB  
- **Database:** MongoDB Atlas  

## Installation
##FOR BACKEND:

cd server

npm install.


##FOR FRONTEND:

cd client

npm install


##START BACKEND:

cd server

npm start


##START FRONTEND:

cd client

npm start
